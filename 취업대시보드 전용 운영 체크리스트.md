# 취업대시보드 전용 운영 체크리스트
> 목적: 주식대시보드/취업대시보드 파일 혼합, JSON 충돌, 잘못된 복사/배포를 다시는 만들지 않기 위한 고정 체크리스트

---

## 0. 절대 원칙
1. **취업대시보드 폴더에서만 작업한다.**
   - 작업 폴더: `C:\Users\solom\OneDrive\바탕 화면\job-dashboard`

2. **와일드카드 복사 금지**
   - 금지 예시: `index*.html`, `app*.js`, `styles*.css`
   - 항상 **정확한 파일명만** 복사한다.

3. **프론트 3개는 항상 한 세트로 교체**
   - `index.html`
   - `web/app.js`
   - `web/styles.css`

4. **JSON 충돌 마커가 있으면 배포 금지**
   - 검사 문자열:
     - `<<<<<<<`
     - `=======`
     - `>>>>>>>`

5. **로컬 확인 후 정상일 때만 GitHub 반영**
   - 설명 후 바로 수정파일 제공 → 파일 교체 → 로컬 실행 → 로컬 사이트 1차 확인 → 정상일 때만 GitHub 반영

## 4. 빌드 후 즉시 검사
### 4-1. JSON 문법 검사
```powershell
python -m json.tool .\dashboard_index.json > $null
```

정상 기준:
- 에러 없이 끝나야 함

### 4-2. 안양/군포/과천 포함 여부 검사
```powershell
Get-Content .\dashboard_index.json -Encoding utf8 | Select-String '"region": "안양"','"region": "군포"','"region": "과천"'
```

정상 기준:
- 안양/군포/과천 문자열이 나와야 함

### 4-3. JSON 충돌 마커 검사
```powershell
Select-String -Path ".\dashboard_index.json",".\data\*.json" -Pattern "<<<<<<<","=======",">>>>>>>"
```

정상 기준:
- **아무 결과도 안 나와야 함**

## 9. 문제가 생겼을 때 즉시 보는 포인트
### 증상: 화면은 뜨는데 데이터가 없음
- `dashboard_index.json` 존재 여부 확인
- `python -m json.tool .\dashboard_index.json > $null`
- `Get-Content .\dashboard_index.json | Select-String '"region": "안양"','"region": "군포"','"region": "과천"'`

### 증상: 브라우저 Console에 JSON SyntaxError
- JSON 파일에 충돌 마커 확인
- `Select-String -Path ".\dashboard_index.json",".\data\*.json" -Pattern "<<<<<<<","=======",">>>>>>>"`

### 증상: 갑자기 주식대시보드 화면이 뜸
- `index.html`, `web/app.js`, `web/styles.css` 다시 검사
- 와일드카드 복사했는지 확인
- 백업 파일/다른 프로젝트 파일 혼입 여부 확인
